Платежи

Платформа поддерживает 5 платёжных систем. Оплата и активация подписки происходят автоматически.

Платёжные системы

Каждая платёжная система настраивается в разделе Settings админ-панели. Можно включить любую комбинацию.

СистемаСпособы оплатыНастройки
YooKassa Банковские карты, СБП ID магазина, секретный ключ, email для чека (опционально)
Telegram Stars Stars внутри Telegram Токен бота, курс конвертации рубли/Stars (по умолчанию 2.5)
CryptoBot BTC, ETH, USDT и другие криптовалюты Токен от @CryptoBot, режим testnet (опционально)
YooMoney Кошелёк, карты, переводы Токен, номер кошелька
FreeKassa 60+ способов оплаты, вывод средств ID мерчанта, секретные слова, API-ключ (для выплат)
Telegram Stars

Работает без внешних сервисов — оплата происходит прямо в Telegram через бота. Курс конвертации рублей в Stars настраивается в админке. Пример: при курсе 2.5 и цене 100 руб. пользователь заплатит 40 Stars.

Процесс оплаты

Пользователь выбирает тариф и способ оплаты
В Mini App отображаются только включённые платёжные системы. Пользователь выбирает план, количество устройств (1–10) и опционально вводит промокод.
Система создаёт платёж
Система рассчитывает итоговую стоимость: базовая цена + доплата за устройства − скидка по промокоду. Создаётся платёж со статусом pending. Генерируется ссылка на оплату.
Пользователь оплачивает
Пользователь перенаправляется на страницу платёжной системы (или оплачивает внутри Telegram для Stars). После оплаты платёжная система отправляет вебхук на сервер.
Обработка вебхука
Сервер верифицирует подпись вебхука, проверяет сумму и статус. Используется защита от дублирования. Статус платежа обновляется на completed.
Активация подписки
Если у пользователя уже была подписка — дни добавляются к текущей дате истечения. Если это первая покупка — создаются ключи на все локации. Трафик сбрасывается.
Уведомления
Пользователь получает сообщение в Telegram с подтверждением оплаты. Админ получает уведомление о новом платеже. Данные в Mini App и админ-панели обновляются в реальном времени.

Расчёт стоимости

Формула итоговой цены:

итого = цена_тарифа + (кол_устройств - 1) × доплата_за_устройство - скидка_промокода
Пример

Тариф «3 месяца» (280 ₽) + 3 устройства (доплата 50 ₽/шт) + промокод 10%:

280 + (3-1) × 50 = 380 ₽ → со скидкой 10%: 380 − 38 = 342 ₽

СтатусОписаниеКогда устанавливается

Пробный период

Пробный период активируется через Mini App. Каждый пользователь может активировать его только один раз.

Доплата за устройства

Доплата за устройство (настраивается в админке, по умолчанию 50 ₽) определяет стоимость каждого дополнительного устройства сверх одного. Пользователь может выбрать от 1 до 5 устройств при покупке.

Защита от дублирования

Система предотвращает повторную обработку одного и того же платежа — подписка не будет активирована дважды.

Чеки (Мой Налог)

Платформа поддерживает интеграцию с «Мой Налог» (ФНС) для автоматической генерации фискальных чеков.

Настройки

  • Включить/выключить генерацию чеков
  • Автоматически отправлять чек пользователю в Telegram
  • Название услуги в чеке (по умолчанию «Подписка VPN»)

При успешной оплате система автоматически создаёт чек в ФНС и при включённой настройке отправляет кнопку «Открыть чек» в уведомлении пользователю.
