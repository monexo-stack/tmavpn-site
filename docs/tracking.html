Трекинг и аналитика

Отслеживание рекламных кампаний, атрибуция пользователей, аналитика доходов и конверсии.

Рекламные кампании

Трекинг позволяет создавать уникальные ссылки для каждого рекламного канала и отслеживать эффективность.

Создание кампании

При создании трекинг-ссылки указываются:

  • Название — отображаемое название кампании
  • Идентификатор — уникальный slug, используется в URL
  • Источник — откуда трафик: Telegram, сайт, реклама и др.
  • Тип ссылки — на бота или на Mini App

Генерация URL

URL формируется автоматически на основе типа ссылки. Для бота — ссылка на Telegram-бота с параметром кампании. Для Mini App — ссылка на приложение с параметром кампании. Готовую ссылку можно скопировать в один клик.

Как работает атрибуция

Пользователь переходит по трекинг-ссылке
Счётчик кликов увеличивается. Пользователь попадает в бота или Mini App.
При регистрации сохраняется кампания
Кампания привязывается к профилю пользователя. Связь сохраняется навсегда.
Все действия пользователя атрибуцируются к кампании
Регистрация, покупка подписки, доход — всё привязывается к исходной кампании для расчёта конверсии и ROI.

Метрики кампании

МетрикаОписаниеФормула
RegistrationsЗарегистрированные пользователи—
ConversionsПользователи с оплаченной подпиской—
RevenueСуммарный доход от пользователей кампании—
Conversion RateПроцент конверсии(conversions / registrations) × 100%

Автоматические алерты

Система автоматически предупреждает о проблемных кампаниях:

  • Низкая конверсия — менее 5% при 10+ регистрациях
  • Подозрение на ботов — более 3 аккаунтов с высокой оценкой бота (50+)

Детальная информация по пользователям кампании

Для каждой кампании доступен список пользователей с метриками:

  • Username, полное имя, Telegram ID
  • Статус подписки и текущий тариф
  • Доход от пользователя
  • Время до первой покупки
  • Использованный трафик
  • Количество активных устройств
  • Bot score и флаг подозрительной активности
  • Оценка возраста Telegram-аккаунта (по диапазону ID)

Массовый бан ботов

Для каждой кампании доступна функция Ban Bots — массовый бан пользователей с высокой оценкой бота. Порог задаётся при запуске (по умолчанию 50+).

Данные на графике

Для каждой точки графика отображаются:

  • Регистрации — новые пользователи за период
  • Платные подписки — новые оплаченные подписки
  • Trial-активации — активации пробного периода
  • Доход — сумма успешных платежей в рублях

График обновляется в реальном времени через WebSocket — при новом платеже данные обновляются автоматически. Навигация по периодам через кнопки «Назад/Вперёд» или прокрутку колеса мыши.
