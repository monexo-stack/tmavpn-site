Telegram бот

Основной бот платформы: приветствие, уведомления о подписке, реферальная система. Отдельный бот поддержки: FAQ и обратная связь.

Команды основного бота

/start

Первое взаимодействие пользователя с ботом. Команда выполняет:

  1. Проверяет, не заблокирован ли пользователь
  2. Обрабатывает реферальную ссылку (если есть параметр ref_) или рекламную кампанию
  3. Создаёт нового пользователя или обновляет данные существующего
  4. Отправляет приветственное сообщение с кнопками

Кнопки под приветственным сообщением:

  • Открыть приложение — открывает Mini App
  • Поддержка — ссылка на бота поддержки
  • Реферальная система — показывает информацию о реферальной программе
Приветственное сообщение

Текст и кнопки приветствия полностью настраиваются через админ-панель (Settings → Bot). Поддерживается подстановка имени пользователя в текст.

/help

Показывает справочное сообщение с описанием возможностей бота.

Уведомления о подписке

Бот автоматически отправляет уведомления о состоянии подписки. Проверка выполняется каждые 30 минут.

Пробный период

Для пользователей с пробной подпиской, которые ещё не подключались к VPN:

МоментУведомление
1 день после активацииНапоминание подключиться — «вы активировали пробный период, но ещё не подключились»
3 дня после активацииПовторное напоминание — «пробный период наполовину прошёл, а VPN не подключён»
Последний деньПредупреждение — «завтра пробный период закончится»
После истеченияУведомление об отключении — «пробный период закончился, VPN отключён»

Если пользователь подключался к VPN (использовал трафик), отправляется только уведомление в последний день.

Платная подписка

МоментУведомление
3 дня до конца«Подписка заканчивается через 3 дня»
1 день до конца«Подписка заканчивается завтра»
День истечения«Подписка истекла, VPN отключён»
1–5 дней после«Прошло несколько дней, настройки сохранены»
Защита от спама

Каждое уведомление отправляется ровно один раз — система запоминает последнее отправленное уведомление для каждого пользователя. Пропущенные пользователи и заблокировавшие бота автоматически пропускаются.

Каждое уведомление содержит кнопку действия: «Настроить VPN», «Выбрать тариф» или «Продлить подписку» — при нажатии открывается Mini App на нужной странице.

Реферальная система в боте

Как работает

При нажатии кнопки «Реферальная система» бот показывает:

  • Персональную реферальную ссылку пользователя
  • Количество приглашённых
  • Заработанные бонусные дни (в режиме дней) или баланс в рублях (в денежном режиме)
  • Описание условий программы с разбивкой бонусов по тарифам

Кнопки

  • Поделиться с другом — открывает окно отправки реферальной ссылки контакту в Telegram
  • Назад в меню — возвращает к приветственному сообщению

Уведомление о новом реферале

Когда приглашённый пользователь регистрируется, пригласивший получает сообщение: «Отличные новости! [Имя] присоединился по вашей ссылке. Вы получите бонус, когда он оформит подписку.»

Бот поддержки

Отдельный бот для обработки обращений пользователей. Настраивается в Settings → Bot.

FAQ-меню

При запуске бота поддержки пользователь видит кнопки с частыми вопросами:

  • Как подключиться
  • VPN не работает
  • Проблемы с оплатой
  • Поддерживаемые устройства
  • Низкая скорость
  • Моя подписка (показывает статус)
  • Связаться с поддержкой (начать диалог)

Рекомендуемые VPN-клиенты

В FAQ «Поддерживаемые устройства» бот рекомендует:

Mini App при установке рекомендует v2RayTun (iOS/Android/Windows/Linux) — оба варианта совместимы, так как платформа использует стандартный протокол VLESS.

Админ-команды в боте поддержки

Администратор (указанный в настройках) имеет доступ к дополнительным командам в боте поддержки.

/user

Поиск пользователя по Telegram ID или @username. Показывает карточку с информацией:

  • Telegram ID, username, полное имя
  • Статус подписки (активна/истекла/нет)
  • Название тарифа и дата окончания
  • Количество активных ключей
  • Статус бана и пробного периода
  • Дата регистрации

Кнопки действий:

  • Забанить / Разбанить
  • +30 дней — продлить подписку
  • Открыть в админке — ссылка на страницу пользователя в админ-панели

/stats

Краткая статистика сервиса:

  • Всего пользователей
  • Новых за сегодня
  • Активных подписок
  • Истекающих в ближайшие 3 дня
  • Платежей за сегодня
  • Доход за сегодня

/servers

Статус всех серверов с индикаторами:

  • Имя сервера, локация, IP-адрес
  • Статус: Online / Degraded / Offline / Disabled

Автоматические уведомления администратору

Администратор получает уведомления в бот поддержки при:

  • Подозрительной активности (всплеск регистраций, неудачных платежей, банов)
  • Обнаружении ботов через рекламные кампании
  • Ежедневный отчёт (утром): новые пользователи, доход, подписки, истекающие подписки
