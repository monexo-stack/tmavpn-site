TMAVPN Platform

Готовая платформа для запуска VPN-сервиса через Telegram. Включает Mini App для клиентов, React-админку для управления и серверную часть с полной автоматизацией.

Что это

TMAVPN — это комплексное решение для запуска собственного VPN-бизнеса в Telegram. Платформа объединяет клиентское приложение (Telegram Mini App), административную панель и серверную инфраструктуру в единую систему.

Telegram Mini App
Клиентское приложение внутри Telegram. Покупка подписки, установка VPN, профиль пользователя — всё в одном месте.
Админ-панель
React-приложение для управления пользователями, серверами, платежами, рассылками и настройками сервиса.
Серверная часть
FastAPI backend с PostgreSQL, Redis, интеграцией с 3x-ui панелями и платёжными системами.
Telegram бот
Уведомления о подписке, пробном периоде, оплате. Поддержка клиентов через бота.

Как это работает

Пользователь открывает Mini App в Telegram, выбирает тариф, оплачивает и получает доступ к VPN. Система автоматически создаёт конфигурации на всех доступных серверах и выдаёт ссылку подписки — пользователь вставляет её в VPN-клиент и подключается.

Пользователь открывает Mini App
Через Telegram бота или прямую ссылку. Приложение загружает данные пользователя, тариф и настройки темы.
Выбирает тариф и оплачивает
Доступны 4 платёжные системы: YooKassa (карты, СБП), Telegram Stars, CryptoPay (крипта), FreeKassa. Промокоды и реферальная система.
Система создаёт VPN-ключи
Автоматически на всех доступных локациях. Каждому пользователю — один постоянный токен подписки, который никогда не меняется.
Пользователь подключается
Устанавливает VPN-клиент (v2RayTun, Hiddify, v2rayNG), импортирует ссылку подписки и нажимает «Подключить».

Схема взаимодействия

Клиент взаимодействует с платформой через Mini App в Telegram. Запросы идут через Nginx на FastAPI backend, который работает с PostgreSQL для хранения данных и Redis для кэширования. Backend общается с 3x-ui панелями на VPN-серверах для создания и управления VPN-конфигурациями. VPN Agent устанавливается на каждый сервер и передаёт метрики (CPU, RAM, трафик, активные подключения) через WebSocket в реальном времени.

Технологии

Backend

  • FastAPI — асинхронный API-фреймворк
  • SQLAlchemy 2.0 — ORM с async-поддержкой
  • asyncpg — асинхронный драйвер PostgreSQL
  • Aiogram 3 — Telegram Bot API
  • Alembic — миграции базы данных
  • PyJWT — JWT-авторизация

Frontend

  • React 18 — UI-фреймворк
  • TypeScript — типизация
  • Vite — сборщик
  • Tailwind CSS — стилизация (админ-панель)
  • Radix UI — компоненты (таблицы, модалки, выпадающие списки)
  • Lottie — анимации стикеров в Mini App

Инфраструктура

  • Docker Compose — оркестрация контейнеров
  • Nginx — reverse proxy с SSL
  • Let's Encrypt — автоматические SSL-сертификаты
  • Supervisor — управление процессами внутри контейнера

Интеграции

  • 3x-ui — управление VPN-панелями (VLESS + Reality)
  • YooKassa — приём платежей (карты, СБП)
  • Telegram Stars — платежи внутри Telegram
  • CryptoPay — криптовалютные платежи
  • FreeKassa — дополнительные способы оплаты

Установка

Платформа устанавливается одной командой на сервер с Ubuntu 20.04+ или Debian 11+.

Требования

  • VPS с минимум 1 ГБ RAM и 10 ГБ диска
  • Домен, привязанный к IP сервера (A-запись в DNS)
  • Telegram бот (создаётся через @BotFather)
  • Ваш Telegram ID (узнать через @userinfobot)

Процесс установки

Клонируйте репозиторий
git clone git@github.com:monexo-stack/tmavpn.git /opt/vpn-telegram-app
Запустите установщик
cd /opt/vpn-telegram-app && sudo bash install.sh
Следуйте инструкциям в терминале
Скрипт запросит: домен, тип базы данных (рекомендуется локальная), email для SSL, токен Telegram бота и ваш Telegram ID. Автоматически установит Docker, настроит Nginx, получит SSL-сертификат и запустит все сервисы.
Первый вход в админку
Откройте https://ваш-домен/admin, введите Setup Token (показан в терминале после установки), создайте аккаунт администратора и подтвердите через код в Telegram.
После установки

Установщик выведет важные данные: Setup Token, пароль базы данных, ключи шифрования. Сохраните их — они показываются только один раз.

Возможности платформы

Для пользователей (Mini App)

  • Покупка подписки с выбором тарифа и количества устройств
  • Пробный период (настраивается в админке)
  • Промокоды со скидкой или дополнительными днями
  • Реферальная программа с двумя режимами: бонусные дни или денежное вознаграждение
  • Пошаговая установка VPN на устройство
  • Инструкции для всех платформ: Android, iOS, Windows, macOS, Linux
  • Профиль с историей транзакций

Для администратора (Панель управления)

  • Управление пользователями: бан, продление подписки, назначение тарифа, экспорт
  • Управление серверами: добавление, мониторинг, балансировка нагрузки
  • Ключи: перевыпуск, удаление, детальная информация о подключениях
  • Платежи: история, статистика, детали каждой транзакции
  • Промокоды: создание, лимиты, отслеживание использования
  • Рассылки: массовые уведомления пользователям с фильтрами
  • Аналитика: доход, конверсия, графики
  • Настройки: платёжные системы, тема Mini App, тексты, стикеры
  • Реферальная система: статистика партнёров, выплаты
  • Трекинг: UTM-метки, рекламные кампании

Автоматизация

  • Автоматическое создание VPN-ключей на всех локациях при покупке
  • Автоматическое отключение ключей при истечении подписки (проверка каждые 60 секунд)
  • Уведомления о пробном периоде и истечении подписки через бота
  • Провижининг ключей при добавлении новой локации
  • Балансировка нагрузки между серверами
  • Синхронизация конфигураций при изменении настроек сервера
  • Мониторинг серверов в реальном времени (CPU, RAM, диск, трафик)
  • Автоустановка агента на новые серверы
