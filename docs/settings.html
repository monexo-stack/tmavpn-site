Настройки

Централизованная конфигурация платформы: бот, платёжные системы, тарифы, тема Mini App, тексты и стикеры.

Все настройки управляются через админ-панель. При изменении настроек кэш сбрасывается автоматически.

Безопасность

Чувствительные данные (токены, ключи, пароли) скрыты в интерфейсе. При сохранении без изменения значения оригинальные данные не перезаписываются.

Платёжные системы

Для каждой платёжной системы настраиваются учётные данные и включается/выключается приём платежей.

YooKassa

  • Включение/выключение приёма платежей
  • ID магазина
  • Секретный ключ
  • Email для чека (опционально)

CryptoBot

  • Включение/выключение приёма платежей
  • API-токен от @CryptoBot
  • Использование тестовой сети

Telegram Stars

  • Включение/выключение (по умолчанию включён)
  • Курс конвертации рублей в Stars (настраивается)

YooMoney

  • Включение/выключение приёма платежей
  • OAuth-токен
  • Номер кошелька

FreeKassa

  • Включение/выключение приёма платежей
  • ID мерчанта
  • Секретное слово для формы оплаты
  • Секретное слово для вебхуков
  • API-ключ для выплат

Дополнительно

  • Доплата за дополнительное устройство (настраивается)
  • Тестовый режим платежей (отключает валидацию полей)

Фискальные чеки (Мой Налог)

Платформа поддерживает автоматическую генерацию фискальных чеков:

  • Включение/выключение генерации чеков
  • Автоматическая отправка чека пользователю
  • Настраиваемое название услуги в чеке

Тема Mini App

Полная кастомизация внешнего вида клиентского приложения без изменения кода.

Цветовая схема

ПараметрОписание
Основной цветКнопки, ссылки, акценты — настраивается
Тёмный вариант основного цветаРассчитывается автоматически или задаётся вручную
Цвет успехаДля индикаторов успешных действий — настраивается
Цвет ошибкиДля индикаторов ошибок — настраивается

Фоновые градиенты

Предустановленные варианты фонов для тёмной и светлой темы Telegram. Категории:

  • Градиенты — плавные переходы между цветами
  • Сплошные цвета — однотонные фоны

Настраиваются отдельно для фона страницы и шапки, в тёмном и светлом режиме.

Кастомизация текстов

Приветственное сообщение бота

HTML-сообщение, которое пользователь видит при первом запуске бота. Поддерживает подстановку имени пользователя.

Кнопки бота

Для каждой кнопки в Telegram-боте настраиваются текст, эмодзи и стиль отображения:

  • Открыть приложение — текст, эмодзи, стиль
  • Поддержка — текст, эмодзи, стиль
  • Реферальная ссылка — текст, эмодзи
  • Поделиться — текст, эмодзи

Текст для шаринга

Сообщение, которое пользователь отправляет при нажатии кнопки «Поделиться».

Тексты Mini App

Тексты интерфейса настраиваются через админ-панель. Тексты организованы по группам (страницы, компоненты). Сохраняются только изменённые значения — для остальных используются значения по умолчанию.

Стикеры

Анимированные стикеры (Lottie) используются на страницах Mini App для визуального оформления.

Настраиваемые стикеры

Стикеры можно задать для следующих разделов:

  • Главная страница — приветственный стикер
  • Страница оплаты
  • Страница настройки VPN

Загрузка стикеров

  • Форматы: TGS (Telegram Animated Sticker), PNG, JPG, GIF, WEBP
  • TGS-файлы автоматически распаковываются в Lottie JSON при загрузке
  • Максимальный размер: 5 МБ
  • Тип Hero: встроенная анимация или загруженный стикер

Настройки реферальной системы

Выбор режима реферальной программы и настройка бонусов.

Режим «Дни»

  • Бонусные дни за приглашение (по умолчанию 3)
  • Бонусные дни за покупку — настраивается отдельно для каждого тарифа

Режим «Деньги»

  • Процент комиссии от покупки приглашённого (по умолчанию 10%)
  • Минимальная сумма вывода (по умолчанию 500 ₽)
  • API-ключ FreeKassa для автоматических выплат

Уведомления администратора

Настройка автоматических уведомлений, которые администратор получает в бот поддержки.

УведомлениеОписание
Сервер недоступенАлерт при потере связи с VPN-сервером
Сервер восстановленУведомление о восстановлении работы сервера
Подозрительная активностьВсплеск регистраций, неудачных платежей или банов
Обнаружение ботовБоты в рекламных кампаниях (с настраиваемым порогом)
Новый платёжУведомление при каждом успешном платеже (опционально)
Ежедневный отчётУтренняя сводка: новые пользователи, доход, подписки

Настраивается интервал между уведомлениями (cooldown в минутах). Есть кнопка отправки тестового уведомления.

ИИ-поддержка

Интеграция с LLM для автоматических ответов в боте поддержки.

  • Включить/выключить — ИИ-ассистент в поддержке
  • URL провайдера — OpenAI-совместимый API
  • API-ключ — ключ доступа к LLM
  • Модель — выбор модели (например, GPT-4)
  • Температура — креативность ответов (0.0–1.0)

Режим технических работ

При включении режима тех. работ:

  • Бот отвечает всем пользователям настраиваемым сообщением о технических работах
  • Mini App показывает страницу обслуживания
  • Администратор продолжает иметь доступ ко всем функциям

Домен подписок

Настройка отдельного домена для ссылок подписки (подробнее на странице Ключи и подписки).

  • Ввод и проверка DNS-записи домена
  • Автоматическая настройка Nginx и SSL-сертификата
  • Удаление кастомного домена

Логи

Журнал действий администратора и системных событий.

  • Фильтрация по типу действия и типу сущности
  • Раскрытие деталей каждой записи
  • Пагинация (20 записей на странице)

База данных

Инструменты для управления базой данных.

Мониторинг

  • Статус подключения к базе данных
  • Ping и версия сервера

Миграция

Мастер миграции данных с другого сервера:

  • Подключение к исходной базе (хост, порт, имя, пароль, поддержка SSL)
  • Тест подключения перед миграцией
  • Запуск миграции с верификацией результатов
  • Подтверждение переключения на новую базу
