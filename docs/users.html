Пользователи

Управление пользователями VPN-сервиса: просмотр, фильтрация, бан, продление подписки, экспорт данных.

Обзор

Каждый пользователь, который запускает Mini App или бота — автоматически регистрируется в системе. Ему присваивается уникальный Telegram ID, фиксируется источник регистрации (органика, реферал, рекламная кампания), IP-адрес и платформа.

Пользователь может находиться в одном из состояний:

СтатусОписание
АктивенПодписка действует, ключи включены
ИстёкПодписка истекла, ключи автоматически отключены
ЗабаненАдминистратор заблокировал пользователя, все ключи отключены
Нет подпискиПользователь зарегистрирован, но не покупал подписку

Аналитика

Над таблицей отображаются карточки с метриками:

  • Всего пользователей — общее количество зарегистрированных
  • Платящие — с активной подпиской (не пробной)
  • Пробный период — на пробном тарифе
  • Онлайн — пользователей в приложении прямо сейчас (данные в реальном времени через WebSocket)

Таблица поддерживает поиск и фильтрацию для быстрого нахождения нужных пользователей.

Текстовый поиск по имени, @username или Telegram ID. Работает мгновенно по мере ввода.

Скрыть неактивных

Переключатель «Скрыть неактивных» в панели инструментов скрывает пользователей без подписки. Настройка сохраняется между сеансами.

Действия

Массовые действия

Выделите несколько пользователей чекбоксами — появится плавающая панель с действиями:

  • Забанить — блокирует выбранных пользователей, отключает все ключи
  • Разбанить — снимает блокировку
  • Назначить тариф — создаёт подписку на выбранный тариф для всех выделенных
  • Продлить подписку — добавляет дни к существующей подписке (быстрый выбор: 7, 14, 30, 90 дней или произвольное значение)

Пример: Вы хотите дать бонус активным пользователям. Отфильтруйте по тарифу «6 месяцев», выделите нужных, нажмите «Продлить подписку» → 7 дней. Всем выбранным добавится неделя к текущей подписке.

Действия с одним пользователем

Нажмите на строку пользователя для открытия карточки с полной информацией и действиями.

Экспорт

Кнопка «Экспорт» скачивает CSV-файл с данными пользователей. Доступны три варианта:

  • Все пользователи
  • Только с активной подпиской
  • Только без подписки

Карточка пользователя

При клике на пользователя открывается модальное окно с полной информацией.

Шапка

  • Имя и @username (со ссылкой на Telegram)
  • Telegram ID (кликабельный для копирования)
  • Индикатор онлайна (зелёная точка если активен менее 2 минут назад)
  • Источник регистрации

Кнопки действий

  • Назначить тариф — появляется если у пользователя нет подписки. Выбор тарифа из списка
  • Продлить — появляется если подписка есть. Добавляет дни
  • Забанить / Разбанить — переключение блокировки
  • Удалить — полное удаление пользователя со всеми данными (ключи, платежи, промокоды, рефералы)

Вкладка «Обзор»

Общая информация: суммарный доход, статус подписки, текущий тариф, дата регистрации, последняя активность, количество подписок.

Вкладка «Платежи»

Список всех транзакций пользователя: план, дата, сумма и статус (Успешно, В обработке, Неуспешно, Возврат).

Вкладка «Рефералы»

Пользователи, которые зарегистрировались по реферальной ссылке данного пользователя. Для каждого реферала: имя, дата регистрации, начисленные бонусные дни.

Вкладка «Подписки»

Все подписки пользователя, сгруппированные по токену. Для каждой:

  • Статус (активна / истекла) с датой окончания
  • Оставшиеся дни
  • Токен подписки (кликабельный для копирования)
  • Список локаций, на которых созданы ключи

Жизненный цикл пользователя

Регистрация
Пользователь открывает Mini App или бота. Система фиксирует Telegram ID, имя, источник (реферальная ссылка, рекламная кампания или органика), IP-адрес и платформу.
Пробный период
Пользователь активирует бесплатный пробный период (длительность настраивается в админке). Система создаёт ключи на всех доступных локациях.
Покупка подписки
Пользователь выбирает тариф, оплачивает. Подписка продлевается, ключи активируются.
Использование
Ключи работают, трафик считается. Перед окончанием подписки бот отправляет уведомления (за 3 дня, за 1 день, в день окончания).
Истечение подписки
Ключи автоматически отключаются (проверка каждые 60 секунд). Бот отправляет уведомление о продлении. Через 3 дня — повторное напоминание.

Уведомления

Система автоматически отправляет уведомления через Telegram-бота на ключевых этапах. Уведомления не дублируются — система запоминает последнее отправленное уведомление.

МоментПробный периодПлатная подписка
День 1Приветствие + инструкция—
День 3Напоминание об использовании—
Последний деньПредложение купить—
За 3 дня до окончания—Напоминание о продлении
За 1 день—Напоминание о продлении
ИстеклаПредложение купитьПредложение продлить
3 дня после истечения—Финальное напоминание

Обнаружение ботов

Система оценивает вероятность того, что пользователь является ботом. Каждому пользователю присваивается Bot Score от 0 до 100.

Как работает

При регистрации фиксируются: IP-адрес, User-Agent и платформа. На основе этих данных система вычисляет оценку. В таблице пользователей Score отображается цветом:

  • 0-24 — обычный пользователь
  • 25-49 — подозрительный
  • 50-100 — вероятно бот

Действия администратора

  • Подтвердить как бота — устанавливает Score = 100, банит пользователя и отключает все ключи
  • Фильтр по Score — в таблице пользователей можно отфильтровать ботов (50+) и подозрительных (25+)
